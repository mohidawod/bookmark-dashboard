## ‚ú® Features

- **Professional UI/UX**: Clean, calm, and modern interface with a premium aesthetic.
- **Organization**: Group bookmarks into custom categories with collapsible sections.
- **Smart Icons**: Automatically fetches website favicons and normalizes them for a unified look.
- **Persistence**: Saves all your data locally using browser `localStorage`.
- **Customization**:
  - **Dark Mode**: High-contrast, easy-on-the-eyes theme toggle.
  - **Inline Editing**: Double-click any title or name to rename it instantly.
  - **Pinning**: Pin your most important bookmarks to the top of their categories.
- **Productivity**:
  - **Drag & Drop**: Reorder bookmarks within or between categories.
  - **Real-time Search**: Quick filtering across all items and categories.
  - **Keyboard Shortcuts**: Press `/` to focus search and `Esc` to clear.
- **PWA Ready**: Installable on desktop and mobile with offline support via Service Workers.
- **Data Portability**: Export and import your data as a `.json` file.

## üöÄ How to Use

1. **Add Bookmarks**: Click the "Add Bookmark" button, enter the name, URL, and category.
2. **Edit**: Double-click any bookmark name or category title to edit it inline.
3. **Pin**: Hover over a card and click the pin icon to keep it at the top.
4. **Organize**: Use the drag handle (card body) to move items around.
5. **Themes**: Toggle between Light and Dark mode using the icon in the header.
